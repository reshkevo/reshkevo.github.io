01 — The rule we don't break

Every account we've seen blow up died the same way — not from a bad trade, from an unmanaged one.

Rule 01

Position size before entry

You decide how much you can lose before you decide how much you want to make. Size is set from your stop, never the other way round.

Rule 02

The stop-loss is not optional

Every position ships with an exit already placed. If a setup doesn't have a clean invalidation point, it isn't a setup — it's a hope.

Rule 03

One bad week can't end you

We cap risk per trade and per week so that a losing streak is an inconvenience, not an extinction event for your account.
